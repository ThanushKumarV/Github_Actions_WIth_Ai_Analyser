# 🤖 AI-Powered CI/CD Failure Analyzer

## 🚀 Overview

This project integrates AI into a CI/CD pipeline to automatically analyze build failures and suggest fixes.

## 🧠 How It Works

1. Code is pushed to GitHub
2. CI/CD pipeline is triggered using GitHub Actions
3. Application fails intentionally
4. Error logs are captured
5. Logs are sent to an AI model (LLM)
6. AI returns root cause and fix suggestions

## ⚙️ Tech Stack

* GitHub Actions (CI/CD)
* Python
* OpenRouter (LLM API)
* Bash scripting

## 💥 Example Output

Cause: Division by zero
Fix: Ensure denominator is not zero before division

* Reduces debugging time
* Helps developers quickly identify issues
* Demonstrates AI integration in DevOps workflows
